# Notice

This repository is an independent offline self-study port of the Shark File
System lab shape used in CMU 15-213 / 15-513. It is not an official CMU
release and is not affiliated with Carnegie Mellon University or the course
staff.

Original SFS lab materials belong to Carnegie Mellon University and the
15-213 / 15-513 course staff. The local packaging, checks, diagnostic scripts,
and documentation in this repository are provided to make self-study possible
without CMU-only infrastructure.

Do not publish completed `sfs-disk.c` solutions or use this repository to
distribute answer keys. Public changes should preserve the starter boundary:
`sfs_getpos`, `sfs_seek`, and `sfs_rename` remain student work unless you are
working in a private solution copy.
